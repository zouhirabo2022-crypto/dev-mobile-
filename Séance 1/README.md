# Séance 1 — Android natif Java

Support : `Support_cours1 Mobile Natif Android.pdf`

## TP : imc_app

Calculateur d'IMC en Android natif Java. On saisit le poids (kg) et la taille (cm), l'app calcule l'IMC et affiche la catégorie avec une image.

Catégories : Maigreur (< 18.5) / Normal / Surpoids / Obésité modérée / Obésité sévère (> 40).

Ouvrir `imc_app/` dans Android Studio, sync Gradle, lancer le module `app`.
